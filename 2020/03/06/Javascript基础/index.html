JavaScript

什么是JavaScript

JavaScript 是互联网上最流行的脚本语言，这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

  • JavaScript 是一种轻量级的编程语言
  • JavaScript 是可插入 HTML 页面的编程代码。
  • JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。

HTML中的存在方式

  1. <script></script>标签中

  2. 在<script>的src属性或指定的外部文件中

  3. 在某个HTML事件处理器中，如onclick、onmouseover等

DOM操作

什么是DOM

DOM全称为文档对象模型（Document Object Model）。是访问和操作HTML文档的标准方法，通过利用JavaScript来访问、操作HTML。

通过JavaScript来改变网页内容

  1. 获取HTML元素内容

    • 获取元素：getElementByld()通过ID获取元素

    • 通过innerHTML获取对应的内容

  2. 修改HTML元素内容

    • 先通过getElementById获取元素ID，然后通过innerHTML赋值修改元素内容。

  3. 创建动态效果

    document.write()写入到HTML页面中

  4. 添加页面交互效果

  5. JavaScript常用事件

BOM操作

什么是BOM

BOM全称：浏览器对象模型（Browser Object Model）。它使JavaScript有能力与浏览器进行“对话”。

  1. 让浏览器弹出警告，通常用来证明漏洞存在（XSS）

    • alert()：警告弹窗
    • confirm()：确定弹窗
    • prompt()：提示弹窗

  2. 获取cookie信息

  3. 获取浏览器信息

    屏幕信息(window.)screen

  4. 获取/控制页面信息

    (window.)location
